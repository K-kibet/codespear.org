Introduction - What is jQuery

jQuery is a simple and lightweight JavaScript library that contains a lot of methods that wrap JavaScript code meant to accomplish certain tasks into simpler and much shorter pieces of code. It also simplifies a lot of complex JavaScript parts like AJAX calls and DOM manipulation.

Through jQuery you can do all kind of things, from hiding and showing elements, to changing the CSS of your HTML tags, to creating all kind of small animations that will add visual effects to your website.

jQuery offers a simplified way to navigate through a document and, by using well thought methods, the entire code is easy to read and manipulate.

Bellow you can see an example that shows how you can hide an HTML element using jQuery:

$(document).ready(function(){

   $("idOfTheElement").hide();

});

The main features of jQuery are:

  • HTML/DOM manipulation
  • CSS manipulation
  • HTML event methods
  • Effects and animation
  • AJAX
  • Utilities

Adding jQuery to your website

Adding jQuery to your HTML file is essential to make it work. If you ever added JavaScript to your HTML file, then the process is pretty much the same, the only difference being that you have to "download" jQuery for it to work.

The way you do that is by either literally downloading it the necessariy files from the official jQuery website, or you can use a CDN (Content Delivery Network).

Let's see how you can add jQuery to your HTML file using both methods:

Downloading jQuery into your computer

This method is really simple, and it requires you to go to the jQuery website by clicking on this link and downloading the compressed production jQuery 3.3.1

After you do that, all you have to do is add downloaded file to where your main HTML file is, and then using the HTML code that will add jQuery to your page, like bellow: <head>
   <script src="jquery-3.3.1.min.js"></script>
</head>

Using a CDN (Content Delivery Network)

The other way is a lot simple, and doesn't require you to host jQuery on your computer.

Both Google and Microsoft host jQuery, so you can use both links from the sections bellow.

Just add the following code to your HTML page and jQuery should work.

Be aware that in some cases, you may have to add jQuery at the end of your HTML file instead of at the beginning. Simply take the "script" code featured bellow, and add it at the end of your HTML document (before your "body" tag ends).

jQuery Syntax

The jQuery syntax is made mainly to select HTML elements and perform some action on them.

For example, a basic example of the jQuery syntax would be: $(selector).action()

  • A $ sign to define/access jQuery
  • A (selector) to "query (or find)" HTML elements
  • A jQuery action() to be performed on the element(s)

Examples:

  • $(this).hide() - hides the current element
  • $("p").hide() - hides all <p> elements
  • $(".test").hide() - hides all the elements with class="test"
  • $("#test").hide() - hides the element with id="test"

This makes the syntax really easy to remember and really easy to use. Selecting elements is really easy and you can choose whether to select a specific element from your page by using ID and CLASS, or if you want to selectg all of them by simply select a "p" or a "div" element.

However, to use jQuery, everything has to be wrapped into a document.ready event. This event makes sure that jQuery will start running once the page is loaded. It looks like this:

$(document).ready(function(){
// jQuery methods go here...
})

jQuery Effects - Hide/Show

One of the most basics jQuery effects that you can add to your HTML elements is the hide/show effect.

To use it all you have to do is to use jQuery to create a function that will hide or show an element if you click on another element, like a button or a paragraph.

Here's how the code would look like, and an exaplanation for what it does:

$(document).ready(function(){
  $(".hide-button").click(function(){   
    $("p").hide();   
  });
});

$(document).ready(function(){
  $(".show-button").click(function(){   
    $("p").show();   
  });
});

$(document).ready(function(){
  $(".toggle-button").click(function(){   
    $("p").toggle();   
  });
});
  • The first code selects an element with the class of "hide-button", and creates a function: if the user clicks the button, then all the "p" elements within the document will be hidden.
  • The second code selects an element with the class of "show-button", and creates a function: if the user clicks the button, then all the "p" elements within the document will be shown.
  • The third code selects an element with the class of "toggle-button", and creates a function: if the user clicks the button, then all the "p" elements within the document will be hidden or shown, with each click. If the element is hidden and the user clicks, the element will appear, and vice versa.

jQuery Effect - Fade

The "Fade" effect is similar to the Hide/Show effect we talked about above, the main difference being that when you use "Fade" instead of "Show" the transition in which the element goes from hidden to visible is slow and animated. Same thing when an element goes from visible to hidden.

One thing that should be noted is the fact that using "Fade" has the same effect as using "Show" in jQuery. The only difference is that the transition is slower, but the end result is that the element will be visible.

Another thing that should be noted is the parenthesis after the name of the method. By inserting a value inside the parenthesis, you can control how much time it will take the fade animation to be completed.

The only exception is that when it comes to the "fadeTo()" method, the numbers within the parenthesis can also represent the opacity of the element. You can add both a speed and an opacity by adding two values, like this: fadeTo("slow", 0.7);

There are four main fade methods that you can use to show an element:

  • fadeIn() - this method makes an element visible using a short "fade in" animation
  • fadeOut() - this method makes an element invisible using a short "fade out" animation
  • fadeToggle() - this method makes an element either visible or invisible when clicking the same element
  • fadeTo() - this method allows you to hide or show an element by changing its opacity

All methods are used the same way when accessing an element, as shown in the sections bellow:

$(document).ready(function(){
  $("button").click(function(){
    $("element").fadeIn():
  });
})

$(document).ready(function(){
  $("button").click(function(){
    $("element").fadeOut():
  });
})

$(document).ready(function(){
  $("button").click(function(){
    $("element").fadeToggle():
  });
})

$(document).ready(function(){
  $("button").click(function(){
    $("element").fadeTo():
  });
})
